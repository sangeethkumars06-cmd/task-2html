# Laundry Wallah / Laundry Mart - HTML Assignment

## Description

This project is a simple HTML webpage developed as part of the TuteDude MERN Stack Task 2 assignment. The webpage represents an online laundry service and includes sections such as a welcome message, about us, services, price list, booking form, and contact information.

## Features

* Welcome Header
* About Us Section
* Laundry Service Image
* Services List
* Price List Table
* Book Now Form
* Footer with Contact Information

## Technologies Used

* HTML5
